Rich signals

Coarse signals (tab audibility, window focus) work out of the box and require no host permissions beyond the local daemon. Rich signals (playing media elements, document focus, fullscreen / Picture-in-Picture, last user interaction) need a content script on every page. Opt-in only.

Status
—
Permission scope is intentionally narrow. The extension only reads tab/window state and a handful of DOM-level signals (visibility, focus, media playback, user interaction). It never sends page content, URL paths, cookies, or form data. See SECURITY.md and extension/README.md for the full threat model.
